# PetDaDuck2 | Unity Clicker Game with Laravel 
Creating a backend for [PetDaDuck 2](https://github.com/SamEBaker/PetDaDuck2.0) using Laravel and AWS
- Maintaining leaderboard database showing players username and high score
- Login/Registration and Saving + Loading - Players can choose to start fresh or login and continue where they left off, players data is saved periodically and when player quits the game.
- Web requests for saving score, high score, current sprite selection, and score increase amt.
